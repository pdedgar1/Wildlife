# Tropical Wildlife — Interdisciplinary Teaching Resource

31 field photographs from tropical Central America and the coast, built into two interactive teaching tools. Hosted via GitHub Pages.

## Tools

**[Workshop gallery](workshop_gallery.html)** — editorial gallery organised around three cross-cutting patterns (*hide or advertise*, *the night shift*, *where does nature end?*). Images are plotted on a relationship map by signal strategy and habitat modification, with drawn connections between same-species pairs and mimicry rings. Includes quiz mode and interdisciplinary discussion notes per image.

**[Wildlife database](tropical_wildlife_database.html)** — searchable, filterable catalog with species identification, taxonomic classification, habitat data, and field notes.

## GitHub Pages setup

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch → main → / (root)**
4. The site will be live at `https://<your-username>.github.io/<repo-name>/`
